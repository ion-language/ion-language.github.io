The Ion Programming Language

Welcome to Ion, a modernized programming language inspired by C. Experience the power and efficiency of C with a refreshing syntax and enhanced features.

Features

  • Order-independent declarations
  • No header files; instead, a package system where a folder is a package
  • Modernized syntax: func main(argc: int, argv: char const*) : int {}
  • Dynamic arrays and other improvements

Sample Code


func main(argc: int, argv: char const*) : int {
    print("Hello, Ion!");
    return 0;
}
